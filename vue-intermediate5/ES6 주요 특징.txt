1. {} 단위로 변수의 범위가 제한됨.
- 함수든, 객체든.

2. const로 지정한 값 변경 불가능.
단, 객체나 배열의 내부는 변경할 수 있다.
const a = 10;
a = 20; -> error

const a = {};
a.num = 10; -> ok

const a = [];
a.push(20); -> ok

3. let 선언한 값에 대해 재선언 불가능.
- 객체나 함수 안에서.

4. Arrow Function
- function을 =>로 바꿈.
- 콜백 함수 문법을 간결화.

ex)
var arr = ["a", "b", "c"];
arr.forEach(function(value) {
  console.log(value);
});

let arr = ["a", "b", "c"];
arr.forEach(value => console.log(value));

*글로벌 변수는 window 객체에 속한다.
