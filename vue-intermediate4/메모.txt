vue create vue-todo
cd vue-todo
npm run serve

프로젝트 만드는 방법
vue init webpack-simple vue-todo
설정 후 
cd vue-todo
npm i
서버 구동 : npm run dev

vetur(비터) : 싱글 파일 컴포넌트 만들어주는 플러그인
- xxx.vue 파일 만들고 vue 입력후 [tab] 입력

컴포넌트 추가하는 과정
- import ... from ...
- components에 추가 (컴포넌트 태그명 : 컴포넌트 변수)
- 상위 컴포넌트 탬플릿에 컴포넌트 태그명으로 추가
*같은 단어 선택 : Ctrl + d
