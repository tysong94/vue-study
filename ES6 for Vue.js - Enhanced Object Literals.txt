1. 객체 리터럴(데이터)에서 속성 메서드의 축약
var dictionary = {
  lookup: function() {
    console.log('lookup');
  }
}

var dictionary = {
  lookup() {
    console.log('lookup');
  }
}

2. 객체 리터럴에서 속성명의 축약
- 객체의 속성명과 값명이 동일할때 아래와 같이 축약.
var figures = 10;
var dictionary = {
  // figures: figures
  figures
}
